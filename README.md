# Cup & Leaf

Небольшое приложение на Django для публикации обзоров чая.

## Быстрый старт

1. Скопируйте файл `.env.example` в `.env` и при необходимости измените значения.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Выполните миграции командой `python cup_and_leaf/manage.py migrate`.
4. Запустите сервер: `python cup_and_leaf/manage.py runserver`.

## Запуск тестов

Все тесты запускаются командой:

```bash
python cup_and_leaf/manage.py test
```
